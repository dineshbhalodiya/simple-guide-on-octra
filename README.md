# simple-guide-on-octra
simple guide on octra

1st Wallet generation Guide

go to https://github.com/octra-labs/wallet-gen

✅Click on fork

✅Select codespace

⏳Wait for codespace to load

✅Now copy Steps one by one to codespace

🔹 Step 1:

````bash
curl -fsSL https://bun.sh/install | bash
source ~/.bashrc
bun --version

````

🔹 Step 2:

````bash
bun install

````

🔹 Step 3:

````bash
bun run build
````

🔹 Step 4:

````bash

bun start

````
